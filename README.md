# Winter's Blog

个人博客，基于 [Astro](https://astro.build/) 和 [Typography](https://github.com/moeyua/astro-theme-typography) 主题构建。

## 本地开发

```bash
pnpm install
pnpm dev
```

## 添加文章

在 `src/content/posts` 目录下创建 markdown 文件：

```md
---
title: 文章标题
pubDate: 2024-01-01
categories: ["分类"]
description: "文章描述"
---
```

## 部署

通过 GitHub Pages 自动部署。
